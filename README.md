# wafaz
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/fazilvk786/wafaz
> cd wafaz
> npm install
```
###### Run
```bash
> node index.js [<session name>] (session name is optional)
```
